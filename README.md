  <h1>Dodge Ball Game</h1>

  <p>This is a Python game created using Pygame where the player controls a ball to dodge incoming enemies. The game features different shapes, such as circles and squares, and includes difficulty levels. The player must dodge enemies to accumulate points and win the game.</p>

  <h2>About</h2>

  <p>The Dodge Ball Game is built using Pygame, a popular library for creating 2D games in Python. The game includes various features such as collision detection, different shapes, and difficulty levels. Players can restart the game or quit at any time.</p>

  <h2>Screenshots</h2>

  <h2>How to Play</h2>

  <p>Run the Python script to start the game. Use the mouse to control the player's ball and dodge incoming enemies. The game includes difficulty levels (Easy, Medium, Hard, Impossible) that can be selected at the start screen.</p>

  ```python
    python dodge_ball_game.py
  ```

  <h2>Controls</h2>

    - Move the ball with the mouse.
    - Press 'R' to restart the game.
    - Press 'Q' to quit the game.

  <h2>Dependencies</h2>

  <ul>
      <li>Pygame</li>
